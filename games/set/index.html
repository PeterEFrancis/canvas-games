Each card has 4 characteristics (color, number, shape, pattern) and each characteristic has three variations (e.g. red, blue, and green; diamond, oval, squiggle). A SET consists of three cards that are either all the same or all different in each category. Click three cards to make a SET and remove them from the board. If you cannot find a SET or believe there is not one on the board, add 3 cards.
